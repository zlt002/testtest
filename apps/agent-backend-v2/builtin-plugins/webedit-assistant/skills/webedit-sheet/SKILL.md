# WebEdit Sheet

适用场景：当前会话需要在 WebEdit 在线表格里读取、写入、美化、填充、校验或分析单元格和区域。

核心流程：

1. 先列工具：调用 `mcp__browser_extension__list_website_tools`，确认存在表格相关 `webedit_*` tools。
2. 先读上下文：优先读取当前工作表、当前选区、活动单元格、区域尺寸。
3. 判断风险：区分单格、矩形区域、多选或不稳定选区；不稳定时不要做 destructive 操作。
4. 再执行写入或样式化：优先用高阶表格工具，不要自己臆造坐标。
5. 写后回读：再次读取目标区域，核对值、公式或样式是否生效。

安全约束：

- 任何批量写入前都要确认 `rowsCount` 和 `columnsCount` 或等效维度信息。
- 选区识别失败时，不要继续清空、覆盖或批量填充。
- 用户只说“美化一下”时，应先读取表头与选区，再决定边框、对齐、颜色等动作。

输出风格：

- 先交代当前选区和工作表信息。
- 写后反馈回读结果，让用户知道哪些单元格已被实际修改。
