# Windows Lite 更新说明

## 本次更新

- 优化页面捕获对本地 `file://` 样式表的读取能力，保存本地快照时能更完整带出本地 CSS 内容。
- 提升复杂表格页面的捕获稳定性，针对 `vxe-table`、`el-table` 的固定列、横向滚动和隐藏布局单元格做了专项兼容，减少截图/快照错位。
- 优化页面编辑底栏交互：尺寸调整改为更直接的宽高输入，Flex 布局面板补齐主轴/交叉轴和换行控制，常用排版与间距操作更集中。
- 调整页面编辑模式切换反馈，进入/退出编辑时会更早恢复可点击状态，减少等待感。
- 首屏启动改为“先检查模型配置，技能后台继续同步”，只要检测到可用模型就能更快开始对话，不必一直卡在同步前置阶段。
- 新增技能后台同步状态提示；同步失败时可直接在聊天页重新同步，不用重开侧边栏。
- 优化模型不可用时的提示链路：部分运行报错会直接给出去模型配置页的入口，减少手动排查路径。
- 修复工作区路由定位时的目录回退问题，按目标项目/目录打开会话与文件面板时更稳定。

## 升级提示

- 如果你经常保存本地页面快照，这次版本对本地样式文件、固定列表格和横向滚动区域的还原会更稳定，建议优先用新包复测。
- 页面编辑底栏现在更偏向“直接输入和直接切换”，尺寸、Flex、间距等调整会比之前少一层跳转。
- 侧边栏现在可能会先允许你开始对话，再在后台补完技能同步；如果下方出现“技能同步中/失败”提示，按提示等待或重试即可。
